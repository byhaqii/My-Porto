# Personal Portfolio (React + Tailwind)

A modern single-page personal portfolio website with a dark black + purple futuristic style.

## Stack

- React (Vite)
- Tailwind CSS
- Framer Motion
- Lucide React

## Run Locally

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

## Project Structure

```text
src/
 ├── components/
 │    ├── Navbar.jsx
 │    ├── Hero.jsx
 │    ├── About.jsx
 │    ├── Portfolio.jsx
 │    ├── Skills.jsx
 │    └── Footer.jsx
 ├── pages/
 │    └── Home.jsx
 ├── data/
 │    └── portfolioData.js
 ├── App.jsx
 └── main.jsx
```

## Easy Content Editing

- Update section text inside each file in `src/components/`.
- Add or edit projects in `src/data/portfolioData.js`.
